Shoot 'Em Up

OVERVIEW

Shoot 'Em Up is a shooter genre game developed using several classes and OOP principles to generate the automated movement of the CPU enemies and bullets. I was able to implement a GUI and various methods to keep track of player lives, points, and an end game scenario. The game was also develpped with an algorithm prompting the enemy bots’ bullets to follow and target the player’s movement.
